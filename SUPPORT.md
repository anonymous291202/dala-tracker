# Support

Thank you for using Dala Tracker.

If you need help, have questions, or encounter an issue, you can reach out through the following channels.

## Bug Reports

Please use the GitHub Issues page to report bugs or unexpected behavior.

When reporting an issue, include:

- Dala Tracker version
- Windows version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots or screen recordings (if available)

## Feature Requests

Have an idea to improve Dala Tracker?

Open a Feature Request on GitHub with as much detail as possible.

## Response Time

Support requests are reviewed as time permits. While responses may not be immediate, every report and suggestion is appreciated.
