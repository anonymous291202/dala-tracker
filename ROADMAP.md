# Roadmap

This roadmap outlines the planned direction for Dala Tracker. Features and priorities may change as development progresses.

---

## Completed

- ✅ Professional landing website
- ✅ Desktop application foundation
- ✅ Batch head tracking workflow
- ✅ Modern user interface
- ✅ GitHub repository and documentation
- ✅ Automated asset generation pipeline

---

## In Progress

- 🚧 Tracking performance improvements
- 🚧 Better export workflow
- 🚧 Improved user experience
- 🚧 Stability and bug fixes

## Planned

### Tracking

- Automatic tracking point detection
- Multi-object tracking
- Improved tracking accuracy
- Tracking presets

### Performance

- GPU acceleration
- Faster batch processing
- Reduced memory usage
- Background processing

### Export

- Additional export formats
- Better After Effects integration
- Export presets
- Batch export improvements

### User Experience

- Project management
- Recent projects
- Custom preferences
- Keyboard shortcuts
- Theme customization

### Platform

- Windows improvements
- macOS support
- Linux support (under evaluation)

---

## Long-Term Vision

Dala Tracker aims to become a professional desktop solution for video editors by providing fast, reliable, and efficient motion tracking workflows while reducing repetitive manual work.
